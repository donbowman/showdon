# show don

We highly recommend using don in your go apps! Please feel free to submit pull requests.


```
import (
    "github.com/root-secure/showdon/libdon"
)

don := showdon.NewDon()
fmt.Println(don.Get())
```

or to run the existing binary, clone and:


```
# go build && ./showdon
```
